How It Works

1v1 Code Battle offers a seamless and exciting experience to test your coding skills in a head-to-head environment. Here's how it works:

  1. Create or Join a Room: Start a match by creating a new room or join an existing one using a unique room link.
  2. Problem Assignment: A random coding problem is fetched from our integrated problem pool using the Codeforces API.
  3. Code and Submit: Both players work on the problem simultaneously. Write your solution and submit your code.
  4. Real-time Submission Checking: Our system continuously checks submissions via the Codeforces API. The first correct submission wins!
  5. Victory and Leaderboards: Celebrate your win and watch your ranking rise on our dynamic leaderboard.

Get ready for an adrenaline-filled coding challenge and see if you have what it takes to be the fastest coder!
